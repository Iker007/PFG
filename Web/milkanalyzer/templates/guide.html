Guia de uso


Primeros pasos


Antes de nada, si no conoce el objetivo de esta página, se recomienda que acceda a la sección sobre el proyecto en la que se especifica el motivo de la herramienta.

Para poder hacer uso de esta herramienta, es necesario poseer una cuenta para la misma. Esto puede hacerse desde el apartado registrarse. En caso de contar ya con una cuenta, o tras haberse registrado, es necesario iniciar sesión para poder visualizar todas las opciones de las que dispone la herramienta.

Una vez haya iniciado sesión, podrá acceder a su perfil donde podrá revisar y modificar su información personal. Además de esto, obtendrá acceso a la lista de modelos disponibles en la herramienta si accede a la página principal.




Uso de modelos


Una vez situados en la página principal, se mostrará la lista de modelos disponibles para realizar predicciones. Cada modelo tiene características diferentes, las cuales se resumirán en su descripción. Para acceder a un modelo y obtener más información o seleccionarlo para realizar predicciones, basta con hacer click en su nombre.



Dentro de la página de un modelo, se nos mostrarán varias secciones. Antes de nada, en la parte superior de la página, se nos mostrará la información básica del modelo seleccionado junto a 2 botones.


El primer botón nos llevará a un cuaderno de google colab en el que se mostrará el código de la creación y entrenamiento de dicho modelo, con algunas explicaciones para una mejor comprensión. El segundo botón nos permitirá hacer uso del modelo para realizar predicciones, proceso el cual se explica más adelante. Finalmente, cada modelo cuenta con una sección llamada "lista de valores requeridos para el modelo", la cual indica los datos que se han utilizado para el entrenamiento del modelo. También es posible obtener más información sobre estos datos accediendo a sus páginas individuales desde este apartado.



Una vez se selecciona el modelo y se pulsa el botón de usar modelo, pasamos a un formulario en el que se pide especificar el nombre del fichero resultante con las predicciones, además de el archivo csv con las muestras sobre las que predecir. Sólamente se aceptan archivos csv y además estos deben contener muestras con por lo menos los datos especificos para cada modelo (los que se indican en la sección "lista de valores requeridos para el modelo"). Es posible proporcionar más datos de los especificados, pero estos serán ignorados a la hora de realizar predicciones.



A continuación se proporciona una plantilla para asegurarse de que no se produzca ningún error relacionado al formato del documento.

Plantilla

Una vez subido el archivo con las muestras y seleccionado confirmar, la herramienta comenzará a realizar predicciones, (el tiempo varía dependiendo del modelo seleccionado) y una vez finalizado, se guardarán los resultados en la sección predicciones, en dónde se podrán consultar la fecha y hora de dichas predicciones, además de acceder al archivo csv con los resultados y a un reporte en formato pdf con información relevante.
